INTRODUCTION
------------

This module allows you to specify a custom quality on individual image styles.
This is useful when you have very large styles which you may want to reduce
the quality of for performance reasons.

REQUIREMENTS
------------

No special requirements.

INSTALLATION
------------

Install as you would normally install a contributed Drupal module. Visit:
https://www.drupal.org/docs/8/extending-drupal-8/installing-drupal-8-modules
for further information.

CONFIGURATION
-------------

* Configure in Administration >> Configuration >> Media >> Image Styles

  - When editing or adding an image style, under Effect, use the drop-down labelled "Select a new effect" to select Image Style Quality.

  - Type the percentage that you deem necessary and click "Add effect".
